This is a simple spaceship game project developed using the Pygame library.

## Usage

```shell
python main.py
```

## Controls

- Yellow Spaceship:

  - Move Left: A
  - Move Right: D
  - Move Up: W
  - Move Down: S
  - Shoot: Left Control (LCTRL)

- Red Spaceship:

  - Move Left: Left Arrow
  - Move Right: Right Arrow
  - Move Up: Up Arrow
  - Move Down: Down Arrow
  - Shoot: Right Control (RCTRL)
